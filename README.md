# Yogi Exchange

## Development

Create `.env` file to change environment variables. Default values are:

```
APP_CHAIN_ID=1
APP_GAS_PRICE=5000000000
APP_MAX_POOLS=4
APP_POOLS=https://bsc.pools.yogi.fi
APP_FARMS=https://bsc.farms.yogi.fi
```

Run & build:

```
npm install
npm run serve
npm run build
```
